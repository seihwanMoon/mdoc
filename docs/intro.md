# mdoc 소개

이 저장소는 **Obsidian 편집 + GitHub 관리 + Docusaurus 배포**를 전제로 한 공개 문서용 Vault입니다.

## 구조

- `docs/`: 공개 문서 본문
- `static/img/`: 사이트 공용 이미지
- `src/pages/`: 홈 페이지
- `.github/workflows/`: GitHub Pages 배포 설정
